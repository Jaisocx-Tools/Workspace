Tooltip Example

Description

The Tooltip sites ui tool is able to do this:

  • The tooltip is placed near the event target the right way.
  • Easy theming via css variables, and the css file has very few css code lines.
  • The tooltip, and when many tooltips on the page, too, can hide after a timeout.
  • The tooltip class can have other html inside.

Javascript Usage Code Example

Basic usage

import { Tooltip, Constants } from "@jaisocx/tooltip";

const tooltipDarkmode = new Tooltip();
tooltipDarkmode
  .setEventTargetHtmlNodeId( "tooltip-show-button-darkmode" )
  .setCssClasses( "theme-darkmode" )
  .setHtml ( "Hello World!" )
  .render();

With some fine tuning:

import { Tooltip, Constants } from "@jaisocx/tooltip";

const tooltipHideBehaviour = Constants.TooltipHideBehaviour.HIDE_AFTER_TIMEOUT__AND__WHEN_CLICK__ANYWHERE;
const tooltipHideTimoutMilliseconds = 4000;

const tooltipContext = new Tooltip();
tooltipContext
  .setTooltipHideBehaviour( tooltipHideBehaviour )
  .setTimeoutToCloseMillis( tooltipHideTimoutMilliseconds )
  .setEventTargetHtmlNodeId( "tooltip-show-button-context" )
  .setCssClasses( "theme-context" )
  .setIsWithArrow(1)
  .setArrowSize (
    0.73,
    "rem"
  )
  .setHtml ( 
    `
    <ul>
      <li>Context Item 1</li>
      <li>Context Item 2</li>
      <li>Context Item 3</li>
    </ul>
    `
  )
  .setAlignDimensionOneValueOrder (
    [
      //Constants.AlignDimensionOne.BROWSER_TAB_BORDER_TOP,
      Constants.AlignDimensionOne.BROWSER_TAB_BORDER_RIGHT,
      Constants.AlignDimensionOne.BROWSER_TAB_BORDER_LEFT,
      Constants.AlignDimensionOne.BROWSER_TAB_BORDER_BOTTOM,
    ]
  )
  .setTooltipAlignDimensionTwo(Constants.AlignDimensionTwo.EVENT_TARGET_START)
  .setTooltipPaddingAlignDimensionTwo (
    1,
    Constants.CssSizeDim.REM
  )
  .render();
